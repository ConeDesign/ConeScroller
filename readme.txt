CONE SCROLLER v1.0

--USAGE--
Simply add 'scroll-to' class to your 'a' element, then add 'data-scroll' attribute where you put ID of element to which it will scroll.

--EXAMPLE--
<a href="#!" class="scroll-to" data-scroll="#your_element_id">Take me there</a>

--NOTE--
Avoid using '#' in 'href' attribute, use '#!' instead so the page wont jump to it's beginning everytime you use this.
